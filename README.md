## お問い合わせ方法

### アプリ内フィードバック（推奨）
アプリ内の **設定 → フィードバック送信** から、バグ報告や機能リクエストをお送りいただけます。

### GitHub Issues
このリポジトリの [Issues](https://github.com/kizzz9/strimo-support/issues) からもお問い合わせいただけます。

## よくある質問（FAQ）

### ゲームについて

**Q: 歩数が反映されません**
A: 設定 → ヘルスケア から歩数の読み取り許可が有効になっていることを確認してください。

**Q: データを別の端末に引き継ぎたい**
A: Apple Sign-In でログインしている場合、同じ Apple ID で新しい端末にログインすることでデータが同期されます。

**Q: 課金アイテムが反映されません**
A: アプリを再起動してください。解決しない場合は、設定 → 購入の復元 をお試しください。

### アカウントについて

**Q: アカウントを削除したい**
A: 設定 → アカウント → アカウント削除 から削除できます。この操作は取り消せません。

**Q: ゲストモードからログインに切り替えたい**
A: 設定 → Apple でサインイン からアカウントを紐付けできます。
